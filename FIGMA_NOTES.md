# Figma notes

The files in /design/figma-export are visual references for OnThe1.
The file OnThe1_raw_figma_code.zip is reference-only and should not be used as the production app foundation unless explicitly needed for visual/component reference.

Use them for:
- layout
- spacing
- typography hierarchy
- button style
- card style
- tab bar style
- screen structure
- copy direction

Do not assume every visual detail is final behavior.

Behavior rules come from OnThe1_Codex_spec.md.

1. OnThe1_Spec.md
